Types of DRAM

SDRAM — Synchronous DRAM (1993)

Before SDRAM, DRAM was asynchronous: the CPU sent an address and waited an unpredictable amount of time for data. Synchronous DRAM introduced a clock signal, allowing the DRAM and the memory controller to operate in lockstep. This made pipelining and burst transfers practical.

Key Innovations

  • Clocked interface (66–133 MHz)
  • Burst mode: transfer N words per command
  • Programmable CAS latency
  • Mode register for configuration

Specs

  • Bus width: 64-bit
  • Voltage: 3.3 V
  • Transfer rate: 133–266 MT/s
  • Peak BW: ~2.1 GB/s
Single Data Rate SDRAM transfers data only on the rising edge of the clock — one transfer per clock cycle. This is the "single" in SDR-SDRAM.

DDR SDRAM — Double Data Rate (2000)

DDR doubled bandwidth without doubling the clock frequency by transferring data on both the rising and falling edges of the clock. A 100 MHz DDR clock yields 200 MT/s (megatransfers per second). DDR prefetches 2 bits per cell per access (2n prefetch).

Key Innovations

  • Double data rate (both clock edges)
  • 2n prefetch architecture
  • Differential clock (CK/CK̄)
  • DQS data strobe for timing

Specs

  • Voltage: 2.5 V → 2.6 V
  • Transfer rate: 200–400 MT/s
  • Peak BW: ~3.2 GB/s
  • Densities: 64 Mb – 1 Gb

DDR2 (2003)

DDR2 doubled the prefetch to 4n, allowing the internal array to run at 1/4 the I/O rate. This enabled higher I/O clock speeds without stressing the core array. Lower supply voltage reduced power.

Key Innovations

  • 4n prefetch (4 bits fetched per access)
  • On-die termination (ODT)
  • Posted CAS additive latency
  • OCD (Off-Chip Driver) calibration

Specs

  • Voltage: 1.8 V
  • Transfer rate: 400–1066 MT/s
  • Peak BW: ~8.5 GB/s
  • Densities: up to 4 Gb

DDR3 (2007)

DDR3 pushed the prefetch to 8n and cut voltage to 1.5 V. It introduced ZQ calibration (dynamic impedance matching) and write leveling to handle signal integrity at high frequencies. DDR3 dominated from 2009–2015 and is still used in embedded and industrial applications.

Key Innovations

  • 8n prefetch architecture
  • Write leveling
  • ZQ impedance calibration
  • Fly-by topology on DIMMs

Specs

  • Voltage: 1.5 V (DDR3L: 1.35 V)
  • Transfer rate: 800–2133 MT/s
  • Peak BW: ~17 GB/s
  • Densities: up to 8 Gb

DDR4 (2012 spec, mainstream 2014)

DDR4 kept the 8n prefetch but reorganized the die into bank groups, enabling much faster back-to-back column accesses across groups (tCCD_S = 4 clocks vs. tCCD_L = 6 clocks within a group). Voltage dropped to 1.2 V. DDR4 is the dominant desktop and server standard through 2024.

Key Innovations

  • Bank groups (2 or 4)
  • Per-DRAM addressability (DDR4 DDP)
  • CRC for command/address bus
  • Parity on CA bus
  • Pseudo Open Drain (POD) signaling

Specs

  • Voltage: 1.2 V
  • Transfer rate: 1600–3200 MT/s (XMP to 5333+)
  • Peak BW: ~51 GB/s (dual-channel)
  • Densities: up to 32 Gb

DDR5 (2020 spec, mainstream 2021+)

DDR5 is the most significant architectural overhaul since DDR1. Key changes include a doubled prefetch to 16n, an on-module voltage regulator (PMIC), on-die ECC for data integrity within the chip, and the channel split from one 64-bit channel to two independent 32-bit subchannels per DIMM.

Key Innovations

  • 16n prefetch (double DDR4)
  • Two 32-bit subchannels per DIMM
  • On-die ECC (ODECC)
  • On-module PMIC (power management IC)
  • Decision Feedback Equalization (DFE)
  • Same Bank Refresh (SBR) to reduce refresh overhead
  • 8 bank groups × 4 banks = 32 banks per die

Specs

  • Voltage: 1.1 V
  • Transfer rate: 4800–8400 MT/s (JEDEC); 12800+ MT/s (OC)
  • Peak BW: ~134 GB/s (dual-channel DDR5-8400)
  • Densities: up to 64 Gb per die
DDR5 Subchannels Each DDR5 DIMM presents two independent 32-bit (+ 8-bit ECC) subchannels to the memory controller. The controller can issue commands to each subchannel independently, effectively doubling command bandwidth compared to DDR4.

LPDDR — Low Power DDR (Mobile)

LPDDR is the mobile-optimized variant of DDR. It prioritizes power efficiency over raw bandwidth through lower voltages, deeper power-down states, and die-level integration (often soldered directly to the PCB as a package-on-package or PoP).

LPDDR5X at 8533 MT/s (used in Snapdragon 8 Gen 3, Apple A17) delivers over 136 GB/s aggregate bandwidth using four 16-bit channels — rivaling server DDR4 at a fraction of the power.

GDDR — Graphics DDR

Graphics workloads demand extreme bandwidth with wide buses. GDDR is purpose-built for GPUs, soldered directly to the graphics card PCB beside the GPU die. It sacrifices latency and uses higher voltages compared to DDR/LPDDR, optimizing instead for raw GB/s.

GDDR6X PAM4 GDDR6X uses PAM4 (Pulse Amplitude Modulation 4) signaling instead of NRZ, encoding 2 bits per symbol by using 4 voltage levels. This doubles the data rate without doubling the clock frequency — at the cost of more complex equalization.

HBM — High Bandwidth Memory

High Bandwidth Memory takes a radically different approach: instead of using a wide parallel bus on a PCB, DRAM dies are stacked vertically using Through-Silicon Vias (TSVs) and placed in the same package as the processor using 2.5D interposer or 3D integration.

This allows an enormous bus width — 1024 bits per stack — at low clock frequencies, achieving massive bandwidth with lower power than GDDR.

Cost HBM is extraordinarily expensive — typically 5–10× the cost per GB of GDDR6. Its use is limited to high-value compute accelerators (AI/ML GPUs, HPC), high-end CPUs (AMD EPYC 9004 with 3D V-Cache adjacent), and specialized ASICs where bandwidth-per-watt justifies the price.
